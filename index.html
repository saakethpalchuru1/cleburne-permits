Fetching permit data from Cityworks

Because this HTML file runs locally (file://), CORS prevents direct calls to cityworks.cleburne.net. Use one of these methods.

Option A — One-click dashboard (recommended)

Drag this button to your bookmarks bar. While logged into Cityworks PublicAccess, click it. A new tab opens with the full dashboard — no file downloads needed.

🔗 Cityworks Dashboard

Option B — Download JSON snapshot

Drag this button to your bookmarks bar. While on Cityworks, click it to download permits.json, then use Load JSON here.

💾 Cityworks → JSON

Option C — Paste in browser console

On the Cityworks PublicAccess tab, press F12, go to Console, paste the script, Enter. It downloads permits.json.
